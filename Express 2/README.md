## REST API

/ - 시작 페이지
/join - 회원가입 페이지
/login - 로그인 페이지
/mypage - 로그인한 유저가 맞을 경우에만 나오는 마이페이지
/write - 글쓰기 페이지
/list - 글 목록 페이지
/edit/:id - 글 수정 페이지
/detail/:id - 상세 글 페이지
